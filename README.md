# 🛡️ SmartShield - Phishing Detection System

## 📌 Overview

SmartShield is a machine learning-based phishing detection system that analyzes both URL and HTML content to detect malicious websites.

---

## 🚀 Features

* URL-based phishing detection
* HTML content analysis
* Browser extension for real-time alerts
* Combined prediction system

---

## 📁 Project Structure

```
backend/
extension/
scripts/
models/        (download separately)
datasets/      (download separately)
```

## 📥 Setup Instructions

### Step 1: Clone repository

```
git clone https://github.com/eng22cs0173/Phishing-Detection-Using-Machine-Learning.git
cd Phishing-Detection-Using-Machine-Learning
```

---

### Step 2: Download required files

* Open the Google Drive link above
* Download the folder (or required files)
* Place them inside your project directory like this:

```
project/
 ├── models/
 ├── datasets/
 ├── backend/
 ├── extension/
```

---

## ▶️ Run the project

```
cd backend
python app.py
```
